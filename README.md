# Roll-a-Ball
Yeni başlayanlar ve merak edenler için Unity'nin "Roll A Ball" örnek uygulamasına birkaç ekleme yaptım.

Skor tabelasının altına süre eklendi. Oyun başladığında süre başlıyor. Bütün kutular toplandığında süre duruyor ve 10 saniyenin altında bitirilip bitirilmediği kontrol edilip farklı mesajlar görüntüleniyor.

Oyun sonuna "Yeni Oyun" ve "Çıkış" seçeneği eklendi.
